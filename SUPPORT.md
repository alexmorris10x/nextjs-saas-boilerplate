# Support

Use [GitHub Issues](https://github.com/alexmorris10x/nextjs-saas-boilerplate/issues) for reproducible boilerplate bugs and focused feature requests. Include the Node version, package-manager version, repository commit or release, steps to reproduce, and relevant redacted logs.

This project is maintained on a best-effort basis with no guaranteed response time. Application-specific architecture, provider accounts, production incidents, data recovery, and deployment operations are outside the support boundary.

Do not post credentials, production data, or vulnerability details publicly. Report security problems through [SECURITY.md](SECURITY.md).
